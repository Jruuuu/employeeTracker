## Installation Commands <a name="installation"></a>  
- INSTALL NODE.JS
- INSTALL GITBASH
- INSTALL MYSQL *Remember Password*
- To Verify MYSQL Install -"mysql -V"
- Run MYSQL Git Bash - "mysql -u root -p"
- Exit - to exit mysql
- Clone Repo
- Git Bash - "npm init -y"
- Git Bash - "npm inquirer"
- Run Command: node index.js
